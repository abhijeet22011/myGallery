# My Gallery 🖼️

A responsive and visually appealing image gallery website built using **HTML** and **CSS Grid**. This project showcases a modern gallery layout with hover effects, smooth transitions, and clean UI design.

## 📌 Project Preview
A stylish gallery webpage displaying multiple images in a grid-based layout with interactive hover animations.

## 🚀 Features
- Responsive image gallery layout
- Built with pure HTML and CSS
- CSS Grid for structured alignment
- Hover effects on images
- Smooth transitions and scaling animations
- Minimal and modern design
- Beginner-friendly project

## 🛠️ Technologies Used
- HTML5
- CSS3
- CSS Grid
- Flexbox
